# Runbooks

## Common Operations

### Database Migrations
```bash
npm run migrate
```

### Rollback Migration
```bash
npm run migrate:undo
```

### Seed Database
```bash
npm run seed
```

### Create Admin User
```bash
npm run script:create-admin
```

### Reindex Data
```bash
npm run script:reindex
```
